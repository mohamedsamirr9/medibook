# MediBook - Healthcare Management System

MediBook is a comprehensive healthcare management platform featuring designed to streamline medical bookings and interactions. The application is built using modern full-stack web technologies, ensuring scalability, clean architecture, and responsive user experiences.

## 🏗️ Technical Architecture & Features

### 🔹 Backend (.NET Web API)
Built with **.NET 8.0** following strict separation of concerns, high maintainability, and clean code practices:
* **ORM:** **Entity Framework Core** with Code-First approach.
* **Database Management:** Features complete Database Migrations and automated **Seed Data** to populate lookups, roles, and initial users upon setup.

### 🔸 Frontend (Angular)
* Built using **Angular** with state-of-the-art component structure.
* Styled with **Bootstrap** for a fully responsive, modern layout compatible with mobile, tablet, and desktop screens.
* Integrated with Angular Services to handle seamless asynchronous REST API calls.
